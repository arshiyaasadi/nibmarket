## Key Visual Elements

### Color Indicators
- **Primary Blue** (`#666CFF`): Selected items, active step, primary actions
- **Success Green** (`#72E128`): Success state, completed steps
- **Text Secondary** (`rgba(color, 0.6)`): Helper text, subtitles
- **Divider** (`rgba(color, 0.12)`): Separators, borders

### Interactive States
1. **Hover**: Card elevates, border color changes
2. **Selected**: Primary border, check icon appears
3. **Disabled**: Reduced opacity, no pointer events
4. **Loading**: Spinner in button, disabled state

### Responsive Behavior
- **Mobile (xs)**: 1 column, stacked buttons
- **Tablet (sm)**: 2 columns for funds
- **Desktop (md)**: 3 columns for funds, side-by-side layout
